## JssValidator

#### Plugin de validação de dados com funções pré definidas retornando um JSON com o resultado da validação e a mensagem de erro


#### Dentre as validações disponíveis estão:

###### Validação de Email

###### Validação de CNPJ

###### Validação de CPF
